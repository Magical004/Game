# Phase 1 Quickstart: Core Features and Enhancements

This quickstart describes the intended local workflow for the planned web client. Adjust commands once the project scaffold exists.

## Prerequisites

- Node.js 20+
- npm 10+

## Install

```bash
npm install
```

## Run (dev)

```bash
npm run dev
```

## Test

```bash
npm run test
npm run test:e2e
```

## Build

```bash
npm run build
```
